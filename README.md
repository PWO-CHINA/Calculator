# 化学计算器工具集 (Chemistry Calculator Toolkit)

综合化学分析实验计算工具集，包含四个专业计算器，支持数据管理、历史追溯和离线使用。

## 📋 功能列表

### 1. 盐酸标准溶液配制计算器
- **功能**：稀释配制与浓度标定
- **应用**：实验室日常HCl溶液准备
- **特性**：
  - 溶液稀释自动计算 (C₁V₁ = C₂V₂)
  - 基准物质标定法（无水碳酸钠、硼砂）
  - 历史记录与CSV导出
  - 浅色/深色主题切换
  - 触摸滑动切换计算模式（移动端）

### 2. 混合碱滴定计算器  
- **功能**：NaOH、Na₂CO₃、NaHCO₃混合样分析
- **方法**：双指示剂滴定法（酚酞+甲基橙）
- **特性**：
  - 自动识别碱类型组合
  - 精确浓度与含量计算
  - 相对平均偏差(RAD)评估
  - 数据表格与统计分析

### 3. EDTA标准溶液标定计算器 ⭐ 新增
- **功能**：锌基准法EDTA浓度标定
- **特性**：
  - **模式A - 锌标准溶液配制**：氧化锌与盐酸反应法
  - **模式B - EDTA浓度标定**：3组平行测定计算
  - 自动浓度同步传递
  - 滴定记录与详情查看
  - 4位有效数字修约(GB/T 8170)
  - 相对标准偏差(RSD)准确性判定

### 4. 铅、铋混合溶液含量测定 ⭐ 新增
- **功能**：EDTA连续滴定法分析
- **原理**：pH控制分步滴定
  - pH≈1：铋(Bi³⁺)滴定
  - pH≈5-6：铅(Pb²⁺)滴定
- **特性**：
  - 移动端卡片式输入（支持左右滑动）
  - 质量分数精确计算
  - 数据异常自动检测与警告
  - 平行测定统计分析(RSD)
  - CSV导出含完整测定详情

## 📐 计算原理

### 有效数字修约标准
遵循 **GB/T 8170** "数值修约规则与极限数值的表示和判定"
- 核心规则：四舍六入五成双
- 应用：4位有效数字（浓度、含量等）

### 统计指标

| 指标 | 定义 | 公式 | 应用 |
|------|------|------|------|
| **平均值** | 多次测量的算术平均 | Σx/n | 最终结果 |
| **RSD** | 相对标准偏差 | (s/x̄)×100% | 精密度评估 (≤0.2%) |
| **RAD** | 相对平均偏差 | (Σ\|xᵢ-x̄\|/x̄)×100% | 相对平均偏差 |
